Web
인터넷
    우리가 사용하는 컴퓨터를 전 세계 규모로 연결한 것
    웹 - 인터넷 표준 문서 시스템

웹의 특징
    전 세계의 컴퓨터를 연결
    HTTP 프로토콜 사용 (Hyper Text Transfer Protocol)
    HTML로 작성된 문서 연결 (Hyper Text Markup Language)
    텍스트, 그래픽, 오디오, 비디오, 프로그램 파일 등 멀티미디어 서비스 제공
반응형 웹(responsive web)
    웹 서버에서 제공되는 정보가 다양한 기기에 맞춰 제공되는 웹 환경
    화면의 사이즈에 따라 웹의 모양이 달라진다. (데스크탑, 스마트폰 등)
    
정보 검색
    - 구글, 다음, 네이버, 야후 등
가상공간
    현실 세계를 웹에 가상으로 구축해놓은 것
    세컨드 라이프, 액티브 월드, 오픈 시뮬레이터 등
온라인 쇼핑
이러닝
웹 표준
    개발자 간 일종의 약속
    누가 개발하든 정해진 규칙을 준수하면 모두 호환되어 편리하게 사용
W3C
    웹에 관련된 기술과 웹 브라우저 사용을 위한 표준안 제정
    웹 개발자나 사용자 간의 정보 공유 및 신기술 개발 등에 기여

마크업 언어
    웹에서 사용되는 문서가 어떻게 구조화되는가를 나타내는 언어
    유형:
        구조적 마크업: 문서의 구성 방식을 표현한 것
        유형적 마크업: 문서를 시각적으로 표현하는 방법
        의미적 마크업: 데이터 내용 자체에 관한 것
HTML5
    W3C와 WHATWG의 두 단체가 협동하여 제안한 세로운 HTML규격
    구조적 설계 지원 - header, footer artice, section, aside, nav 등
    그래픽 및 멀티미디어 기능 강화
    CSS3 지원
    자바스크립트 지원
    다양한 API제공
    모바일 웹 지원

웹 브라우저
    HTML 문서를 읽고 웹 페이지에 정보를 표시하는 소프트웨어
    URL을 통해 접근
    크롬, 파이어폭스, 오페라 등

HTML 편집기 종류
    메모장, 노트패드, 울트라에디트 등
    현재에는 visual studio code를 많이 사용한다.
